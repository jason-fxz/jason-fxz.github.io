SDOI2014 - LIS

发表于2020-12-10|更新于2021-07-04|题解
|阅读量:

LOJ - 2196

题意

给定长度为 NNN 序列 AAA ，序列中的每一项 AiA_iAi​ 有删除代价 BiB_iBi​ 和附加属性 CiC_iCi​。
请删除若干项，使得 AAA 的最长上升子序列长度减少至少 111，且付出的代价之和最小，并输出方案。如果有多种方案，请输出将删去项的附加属性排序之后，字典序最小的一种。

N≤700N\le 700 N≤700

题解

求 LIS 跑一遍 DP 即可，记最大的 DP 值为 mxdpmxdpmxdp ，从 DP 值等于 mxdpmxdpmxdp 的位置倒推回去，按转移关系建边得到一个转移图，显然入度为零的都是 DP 值为一的，出度为零的都是 DP 值为 mxdpmxdpmxdp 的。现在就是要求最小的代价使入度为零的点与出度为 mxdpmxdpmxdp 的点不连通。

那不就是最小割吗。具体建图：

  1. 超级源点 SSS 连向所有 DP 值为零的点，边权 +∞+\infty+∞
  2. 所有 DP 值为 mxdpmxdpmxdp 的点连向超级汇点 TTT 边权 +∞+\infty+∞
  3. 对于所有 dp[u]+1=dp[v],u<v,Au<Avdp[u]+1=dp[v],u\lt v,A_u\lt A_vdp[u]+1=dp[v],u<v,Au​<Av​ ，uuu 连向 vvv ，边权 +∞+\infty+∞
  4. 因为代价在点上，拆点，拆 uuu 为 u,u′u,u'u,u′ ，uuu 连 u′u'u′ 边权为 BuB_uBu​

跑一个最大流可知最小代价。

剩下的便是求方案，这里要求 字典序最小的最小割，涉及到求最小割的可行边 （某条边出现在某个最小割方案中）一条边 (u,v)(u,v)(u,v) 为最小割的可行边的充要条件为：

  1. 该边满流 。
  2. 不存在 uuu 到 vvv 的增广路。

判断 uuu 到 vvv 的增广路 BFS 即可。

将边按照 CCC 升序排序依次判断是否为最小割可行边，若是则加入答案，并需要删除它的影响，需要退流 ：uuu 向 SSS 退流， vvv 向 TTT 退流，倒着跑 Dinic 。并删除 (u,v)(u,v)(u,v) 及其反向边，流量都赋零。
