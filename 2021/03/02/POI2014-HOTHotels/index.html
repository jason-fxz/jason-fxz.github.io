长链剖分优化树形DP板题。

luogu-P5905

给你一颗 nnn 个点的树，求有多少点对 (i,j,k)(i,j,k)(i,j,k) 满足 i,j,ki,j,ki,j,k 两两距离相等。

题解

u,v,wu,v,wu,v,w 两两距离相等只有两种情况，（1） ccc 是 u,v,wu,v,wu,v,w 三点的 LCA\mathrm{LCA}LCA 且 u,v,wu,v,wu,v,w 到 ccc 的距离一样。（2）ccc 是 u,vu,vu,v 的 LCA\mathrm{LCA}LCA 且 www 在 ccc 的子树外，u,v,wu,v,wu,v,w 到 ccc 距离相等。

考虑暴力 DP，设 f(u,i)f(u,i)f(u,i) 表示 uuu 的子树中到 uuu 距离为 iii 的点的个数， g(u,i)g(u,i)g(u,i) 表示 uuu 子树中存在多少点对 (v,w)(v,w)(v,w) （记 c=LCA(v,w)c=\mathrm{LCA}(v,w)c=LCA(v,w)）满足 dis(v,c)=dis(w,c)=dis(u,c)+i\mathrm{dis}(v,c)=\mathrm{dis}(w,c)=\mathrm{dis}(u,c)+idis(v,c)=dis(w,c)=dis(u,c)+i 。

按照普通树形 DP 的套路，当前 uuu 点，vvv 是 uuu 当前访问的子节点，g(u,i),f(u,i)g(u,i),f(u,i)g(u,i),f(u,i) 都是 uuu 和之前子树合并的结果：

  1. ans+=f(v,i)×g(u,i+1)ans+= f(v,i)\times g(u,i+1)ans+=f(v,i)×g(u,i+1)
  2. ans+=g(v,i)×f(u,i−1)ans+=g(v,i)\times f(u,i-1)ans+=g(v,i)×f(u,i−1)
  3. g(u,i)+=g(v,i+1)g(u,i)+=g(v,i+1)g(u,i)+=g(v,i+1)
  4. g(u,i)+=f(u,i)×f(v,i−1)g(u,i)+=f(u,i)\times f(v,i-1)g(u,i)+=f(u,i)×f(v,i−1)
  5. f(u,i)+=f(v,i−1)f(u,i)+=f(v,i-1)f(u,i)+=f(v,i−1)

这样可以做到 O(n2)O(n^2)O(n2) 的复杂度，考虑到 DP 的转移只与深度有关，可以用长链剖分优化到 O(n)O(n)O(n) 。这里有个细节， g(u,i)=g(v,i+1)g(u,i)=g(v,i+1)g(u,i)=g(v,i+1) 对应的指针是向左移动的，分配的空间需要开两倍。然后重儿子除了直接提供当前节点的 f,gf,gf,g 对答案还有 g[v][1]g[v][1]g[v][1] 的贡献。
