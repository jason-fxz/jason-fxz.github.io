## 题解

设：
$$
f(n)=\sum_{i=1}^n\sum_{j=1}^{i-1} [\gcd(i,j)=1] \frac{\cos\frac{2\pi j}{i}+1}{2}
$$

那么答案就是 $f(R+1)-f(L)$  。 

> $\mathrm{\color{black}{h}\color{red}{ehezhou}}$ ：“这你都不会，那你可以滚回去学whk了。” 

$$
\begin{aligned}
g(n)&=\sum_{i=1}^{n} [\gcd(i,n)=1] \frac{\cos\frac{2\pi i}{n}+1}{2}\\
&=\sum_{i=1}^{n} \sum_{d\mid \gcd(i,n)} \mu(d) \frac{\cos\frac{2\pi i}{n}+1}{2}\\
&=\sum_{d\mid n} \mu(d) \sum_{i=1}^n [d\mid i] \frac{\cos\frac{2\pi i}{n}+1}{2}\\
&=\sum_{d\mid n} \mu(d) \sum_{i=1}^{n/d} \frac{\cos\frac{2\pi di}{n}+1}{2}\\
&=\frac{1}{2} \left( \sum_{d\mid n} \mu(d) \sum_{i=1}^{n/d} \cos\frac{2\pi i}{n/d}+ \sum_{d\mid n} \mu(d) \frac{n}{d} \right)
\end{aligned}
$$

考虑右边， $\sum_{d|n} \mu(d)id(\frac{n}{d}) =\varphi(n)$

考虑左边 $\sum_{i=1}^{n'} \;\cos\frac{2\pi  i}{n'}$ 怎么算，联想到[单位根](https://baike.baidu.com/item/%E5%8D%95%E4%BD%8D%E6%A0%B9)  $\omega_n^k=\cos \frac{2k\pi}{n}+\sin \frac{2k\pi}{n}\mathrm{i}$ ，单位根的性质，学过 FFT 应该知道，反正 $w_n^k$ 中的 $k$ 就当成指数看好了。还有 $w_n^0=1,w_n^{n+k}=w_n^k$

考虑单位根的等比数列求和：
$$
\begin{matrix}
S=w_n^1+w_n^2+\dots+w_n^n\\
S\cdot w_n^1=w_n^2+w_n^3+\dots+w_n^{n+1}\\
(w_n^1-1)S=w_n^{n+1}-w_n^1\\
S=\frac{w_n^{1}-w_n^1}{w_n^1-1}\\
S=0
\end{matrix}
$$
也就是说 ：
$$
S=\sum_{i=1}^{n} \cos\frac{2\pi i}{n}+\sum_{i=1}^{n} \sin\frac{2\pi i}{n}\mathrm{i}=0
$$

$$
\sum_{i=1}^{n} \cos\frac{2\pi i}{n}=0,\;\sum_{i=1}^{n} \sin\frac{2\pi i}{n}=0
$$
复数的加法实部和虚部分开考虑就行了，**注意** 别忘了特殊情况 $n=1$ ，为 $\cos 2\pi=1$ ，综上可得：

$$
\sum_{i=1}^{n} \;\cos\frac{2\pi  i}{n}=[n=1]
$$
继续推式子：
$$
g(n)=\frac{1}{2}\left(\sum_{d|n}\mu(d)[n/d=1]+\varphi(n) \right)=\frac{\mu(n)+\varphi(n)}{2}
$$
$f$ 是 $g$ 的前缀和，$n\le 10^8$ 线性筛显然 $\color{blue}{\mathrm{TLE}}$ ，~~除非是松怪~~；上杜教筛就完事了。

## CODE

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
const int N = 10000000;
int p[N],tot;
ll mu[N],phi[N];
bool v[N];
void init() {
    mu[1]=phi[1]=1;
    for(int i=2;i<N;++i) {
        if(!v[i]) p[++tot]=i,mu[i]=-1,phi[i]=i-1;
        for(int j=1;j<=tot&&p[j]*i<N;++j) {
            v[i*p[j]]=1;
            if(i%p[j]==0) {
                mu[i*p[j]]=0;
                phi[i*p[j]]=phi[i]*p[j];
                break;
            } 
            mu[i*p[j]]=-mu[i];
            phi[i*p[j]]=phi[i]*(p[j]-1);
        }
    }
    for(int i=2;i<N;++i) mu[i]+=mu[i-1],phi[i]+=phi[i-1];
}
ll MU(int n) {
    if(n<N) return mu[n];
    ll ret=1;
    for(int i=2,j;i<=n;i=j+1) {
        j=n/(n/i);
        ret-=MU(n/i)*(j-i+1);
    }
    return ret;
}
ll PHI(int n) {
    if(n<N) return phi[n];
    ll ret=(ll)(n+1)*n/2;
    for(int i=2,j;i<=n;i=j+1) {
        j=n/(n/i);
        ret-=PHI(n/i)*(j-i+1);
    }
    return ret;
}
ll F(int L,int R) {
    return MU(R+1)+PHI(R+1)-MU(L)-PHI(L);
}
int l,r;
int main() {
    scanf("%d%d",&l,&r);
    init();
    printf("%.2lf\n",(double)F(l,r)/2.0);
}
```
