### 题意

$n$ 种物品，每种有 $c_i$ 个，体积为 $v_i$，给你一个整数集 $S$ ，从这 $n$ 种物品中选出 $k,k\in S$ 种物品，每种物品 $x$ 个（$1\le x \leq c_i$），问能得到哪些小于 $L$ 的总体积值。

$$
\text{所有输入}\leq 2\times 10^3
$$

### 题解

换一种思路考虑背包，弄一个 ```bitset<N> dp[N]``` ，$dp_{i,j}$ 为 $1$ 表示存在选 $j$ 种物品的方案使得总体积为 $i$ ，注意下标顺序

那么会有以下简单的思路

```cpp
for ( i : 第 i 种物品 )
    for ( j : 总体积为 j )
        for ( k (k>0): 第i种物品用了几个 )
            dp[j] |= ( dp[ j - k * v[i] ] << 1 ) ;
```

这样的方法是 $O(\frac{n^2L^2}{\omega})$ ，对与本题还需要优化（注意 ```bitset``` 支持数组下标访问，可以当做一个二进制使用，就算你左移右移移出了规定的 $N$ 也不会 $\mathrm{\color{purple}{RE}}$ ）

显然前两个循环是优化不掉的，我们考虑简化第三个循环

发现对于一个特定的 $j$ ，能更新它的 $j-k\cdot v_i$ 在模 $v_i$ 意义下与 $j$ 同余，所以我们在脑海中抽象的把与 $j$ 同余的下标拉出来组成一个序列，可以知道能更新 $dp_j$ 的下标是这个序列中某个长度为 $c_i$ 的连续一段。

这是一个区间求或，由于**长度固定**（不足 $c_i$ 按 $c_i$ 看），那么我们每隔 $c_i$ 分一个组，在每个组中维护一个前缀和后缀（相当于分块），每次更新的时候 $O(1)$ 计算出左右端点分别在哪个位置（显然是相邻两块），取后缀和前缀合并一下。

这个预处理虽然是双层循环，但是它总共扫描的个数是 $O(L)$ 的，而后面的合并是 $O(\frac{L}{\omega})$ 的。

总复杂度为 $O(\frac{nL^2}{\omega})$。

### CDOE

```cpp
#include <bits/stdc++.h>
#define bt bitset<N>
using namespace std;
template <typename T>
inline void red(T &x) {
    x=0;bool f=0;char ch=getchar();
    while(ch<'0'||ch>'9') {if(ch=='-') f^=1; ch=getchar();}
    while(ch>='0'&&ch<='9') x=(x<<1)+(x<<3)+(T)(ch^48),ch=getchar();
    if(f) x=-x;
}
const int N = 2005;
bt dp[N],ans,g,pre[N],suf[N];
int n,L,m,s[N],c[N],v[N];
namespace test {
    void Pre(int V,int len) {
        for(int p=0;p<V;++p) {
            // j*V+p j%len=0 为关键点 
            // suf[i] 为 i 到下一个关键点的后缀或
            // pre[i] 为 i 到上一个关键点的前缀或
            for(int j=0;j*V+p<=L;++j) {
                if(j%len==0) pre[j*V+p]=dp[j*V+p];
                else pre[j*V+p]=pre[(j-1)*V+p]|dp[j*V+p];
            }
            for(int j=(L-p)/V;j>=0;--j) {
                if(j%len==0) suf[j*V+p]=dp[j*V+p];  
                else suf[j*V+p]=suf[(j+1)*V+p]|dp[j*V+p];
            }
        }
    }
    void ask(bt &B,int L,int R) {B|=(suf[L]|pre[R])<<1;}
    void solve() {
        dp[0][0]=1;
        for(int i=1;i<=n;++i) {
            Pre(v[i],c[i]);
            for(int j=v[i],lp=0;j<=L;++j,++lp) {
                //  j-v[i]*c[i] ~ j-v[i]
                if(lp>=v[i] && j-lp<c[i]*v[i]) lp=0;
                ask(dp[j],lp,j-v[i]);
            }
        }
        for(int i=1;i<=L;++i) {
            dp[i]&=g;
            if(dp[i].count()>0) printf("%d ",i);
        }
        printf("\n");
    }
}
int main() {
    red(n);red(L);
    for(int i=1;i<=n;++i) red(c[i]),red(v[i]);
    red(m);
    for(int i=1;i<=m;++i) red(s[i]),g[s[i]]=1;
    test::solve();    
}
```
